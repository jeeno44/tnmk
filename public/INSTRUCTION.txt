1) Склонировать последний коммит из ветки master к себе в репозиторий (git pull origin master)
2) Настроить конфигурационный файл .env - база данных - user - password (Если нет файла то скопировать .env.example)
3) Удалить все таблицы из базы данных
4) Выполнить команду для выполнения миграций и посева данных - php artisan migrate --seed
5) Запустить (или настроить) веб сервер / Также можно запустить встроенный в Laravel веб сервер командой - php artisan serve
6) Зайти по адресу сайта / если запущен встроенный веб сервер то адрес - 127.0.0.1:8000
7) Авторизироваться на сайте:
    user - admin@mail.com
    pass - administrator555

Примечания:

1) На странице категорий при поиске не выводятся найденные товары,
   а только отображаются красным цветом те категории в которых эти товары были найдены.
2) На странице "Настройки" нет кнопки "сохранить", сохранение производится автоматически при перетаскивании категории.
3) Вместо вашей вёрстки я воспользовался встроенной вёрсткой Bootstrap.
